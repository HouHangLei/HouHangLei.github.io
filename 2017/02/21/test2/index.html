这是一个简介

经过了昨天踩的大坑小坑，终于搭建好了hexo，域名用的是github的。我的博客地址。

本文主要记录配置过程中所参考的文章以及遇到的问题和最后的解决方案，方便日后查询整理。

基础准备

下载Node.js用来生成静态页面

到Node.js官网下载,有一个推荐用于大多数用户使用和一个包含最新功能的版本，我下载的是推荐用于大多数用户的v6.9.5版本。按照流程安装。

安装Git

安装Xcode就自带Git了，或者去这里下载，安装完成后，打开终端，输入git –version，能查看到版本号就说明安装成功了

创建reposity

在GitHub上创建reposity（没有Github的去注册一个）

安装Hexo

上面的安装完以后，开始安装Hexo。

终端输入npm install -g hexo 执行

如果安装失败，终端重新输入sudo npm install -g hexo 执行

安装成功后，初始化Hexo

首先创建一个文件夹，如my_hexo.然后使用终端命令cd进入my_hexo文件夹

然后执行init命令初始化hexo

至此，全部安装工作已经完成！blog就是你的博客根目录，所有的操作都在里面进行。

生成静态页面

hexo generate（hexo g也可以）

本地启动

启动本地服务，进行文章预览调试，命令：

hexo server

浏览器输入http://localhost:4000，没有问题的话就可以看到效果了
